# Semester 5

## Overview
This repository contains various materials and assignments from different courses undertaken during university studies (semester 5) at Gdańsk University of Technology. Each folder corresponds to a specific subject and includes relevant code, documentation, and reports.

## Courses

### 1. Software engineering
Topics covered:
- **Introduction to Software Engineering:** Scope, motivations, and basic concepts.
- **Software Development Lifecycle:** Key areas and processes.
- **Risk and Social Responsibility:** Related to IT systems.
- **Requirements Engineering:** Elicitation, analysis, specification, validation, and management.
- **Conceptual Modeling & UML:** Using UML for system analysis and design (use cases, class, sequence, state, activity diagrams).
- **System Design:** Architecture, high-level design, and detailed class design.
- **Software Quality:** Object-oriented metrics, SOLID principles, and code quality.
- **Software Reuse & Design Patterns:** Classical patterns and others (web, analysis, architectural).
- **User Interface (UI) Design:** Motivations, techniques, and Nielsen's heuristics.
- **Software Testing:** Concepts, techniques (black-box, white-box), levels, and management.
- **Software Reviews & Inspections.**
- **Deployment, Operation, and Maintenance.**
- **Configuration and Change Management.**
- **Software Lifecycle Models:** Classical and non-classical (e.g., Agile) models and their selection.
- **IT Project Management & Methodologies:** Plan-driven and agile approaches.

### 2. Data warehouses
Topics covered:
- **Business Intelligence (BI) Fundamentals:** Basic concepts and scope.
- **Data Warehouse Construction & Implementation:** Principles and methods for building data warehouses.
- **Data Models for Data Warehousing:** Knowledge and application of specific data models (e.g., star schema, snowflake schema).
- **Data Warehouse Design Process:** From requirements to implementation and optimization.
- **Relational Algebra & Aggregate Functions:** Foundational mathematical knowledge for data processing.
- **OLAP Processing:** Concepts and principles of Online Analytical Processing.
- **Managerial Dashboards:** Creation and use of dashboards for decision support.

### 3. Internet Services Architectures
Topics covered:
- **Java SE Advanced Features:** Practical usage of Stream API, lambda expressions, serialization, and multithreading (custom thread pools).
- **Spring Framework Ecosystem:** Core concepts including Dependency Injection (DI) and Inversion of Control (IoC) using Spring Boot.
- **Data Persistence (ORM):** Database access using Spring Data JPA, Hibernate, and repository pattern implementation.
- **RESTful Web Services:** Designing and implementing REST APIs with Spring MVC, controller architecture, and Data Transfer Objects (DTO).
- **Microservices Architecture:** Decomposition of monolithic applications into microservices, event-based communication, and routing with Spring Cloud Gateway.
- **Frontend Development:** Developing Single Page Applications (SPA) using Angular framework, component-based architecture, and service consumption.
- **Containerization:** Containerizing applications using Docker, configuring NGINX as a reverse proxy, and orchestration with Docker Compose.
